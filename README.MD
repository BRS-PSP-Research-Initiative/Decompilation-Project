## About

Black Rock Shooter: the Game is not legally, sanely available at this time. The developer no longer exists and self-published the game in Japan. This repository contains the decompiled code retrieved from the game's binary in hopes that it will be educational and aid in the preservation of the title. I claim no ownership of the original decompiled code but reserve the right to copyright and license modifications made in order to make the code usable.
